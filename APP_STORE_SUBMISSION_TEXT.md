## 📖 **4. 説明文（4,000文字以内）**

```
アプリをダウンロードする時、利用規約ってちゃんと読んでますか？
賃貸契約の書類、全部理解できてますか？

TrapFinderは、そんな「読むのが面倒な長文」をAIがサクッと読み解いて、重要なポイントだけ教えてくれるアプリです。

■ できること

カメラで撮影するだけで、紙の書類もすぐに読み取れます。PDFファイルやWebページのURLにも対応しているので、オンラインサービスの利用規約もそのまま貼り付けてチェックできます。

AIが文書を読んで、「ここは要注意」「ここは知っておくべき」というポイントを自動でピックアップ。元の文章も一緒に表示されるので、どこに書いてあったのかすぐに分かります。

■ こんな時に使えます

・Webサービスの利用規約やプライバシーポリシー
・アパートやマンションの賃貸契約書
・アルバイトや就職時の雇用契約
・クレジットカードやローンの契約書
・保険の約款や重要事項説明書

難しい言葉が並んでいても、AIが分かりやすく噛み砕いて説明してくれます。

■ プライバシーは守ります

読み込んだ文書は解析が終わったらすぐに削除されます。保存もしませんし、AI学習にも使いません。安心してお使いください。

■ 料金プラン

無料プランでも使えます（広告視聴後、1回10,000文字まで）。

もっと使いたい方には有料プランもあります：
・スタンダード：月額280円で広告なし、50,000文字まで対応
・プロ：月額580円で高性能AI（GPT-4o）を使用、50,000文字まで対応

※このアプリは文書の理解をお手伝いするツールです。法律の専門家ではありませんので、大事な判断をする時は弁護士さんなどにご相談ください。
```

### **メモ（審査員へのメッセージ）**

```
【アプリ概要】
TrapFinderは、利用規約やプライバシーポリシーなど、長くて複雑な文書をAIが読み解いて、重要なポイントを分かりやすく教えてくれるアプリです。法的なアドバイスをするものではなく、文書の理解をサポートするツールです。

【テスト方法】
無料プランから試せます（広告視聴後に利用可能）。以下の4つの方法で文書を読み込めます：

1. カメラスキャン：紙の書類を撮影
2. PDF読み込み：ファイルを選択
3. URL入力：Webページから直接読み込み
   例：https://www.apple.com/legal/internet-services/itunes/jp/terms.html
4. テキスト貼り付け：コピーしたテキストを貼り付け

【サンプルテキスト】
以下をコピーして「テキスト入力」から試せます：
「この利用規約は、ユーザーと当社の間に締結される契約です。本サービスの利用により、ユーザーは本規約のすべての条項に同意したものとみなされます。サービス利用料は月額500円（税込）で、自動更新されます。解約は利用期間終了の24時間前までに行う必要があります。」

【APIキーについて】
OpenAI APIキーは、ビルド時に安全に注入されており、ソースコードには含まれていません。

【広告について】
無料プランでは、Google AdMobのリワード広告を使用しています。広告視聴後にAI解析が実行されます。広告が表示されない場合でも、デバッグ環境では解析を続行できるように実装しています。

【プライバシー】
• ユーザーデータは保存されません
• AI学習にも使用されません
• 解析後、データは速やかに削除されます
• OpenAI APIのデータ使用ポリシーに準拠しています

【サブスクリプション】
• スタンダード: ¥280/月（standard_monthly）
• プロ: ¥580/月（pro_monthly）

StoreKit Configuration Fileを使用してテストできます。

よろしくお願いいたします。
```
