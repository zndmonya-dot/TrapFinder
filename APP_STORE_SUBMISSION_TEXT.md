## 📖 **4. 説明文（4,000文字以内）**

```
TrapFinderは、利用規約やプライバシーポリシーなどの長文をAIが解析し、重要なポイントを抽出するアプリです。

◆ 主な機能

カメラ撮影、PDFファイル、URL、テキスト貼り付けに対応。4つの方法で文書を読み込めます。

AIが文書を自動解析し、注意すべきポイントや重要な情報を抽出。該当箇所の引用とともに分かりやすく表示します。

◆ 対応文書

- Webサービスの利用規約
- プライバシーポリシー
- 賃貸契約書
- 雇用契約書
- クレジットカード規約
- 保険の約款
- その他、理解が必要な文書全般

◆ プライバシー

解析後のデータは保存されません。AI学習にも使用されません。

◆ 料金プラン

無料プラン：10,000文字/回（広告視聴後に利用可能）
スタンダード：月額280円、50,000文字/回、広告なし
プロ：月額580円、高性能AI（GPT-4o）、50,000文字/回、広告なし

◆ 免責事項

本アプリは文書理解を補助する情報提供ツールです。法的助言を提供するものではありません。重要な判断が必要な場合は専門家にご相談ください。
```

【アプリ概要】
TrapFinderは、利用規約やプライバシーポリシーなどの文書をAIが解析し、重要ポイントを抽出する文書理解補助アプリです。法的助言を提供するものではありません。

無料プランで4つの読み込み方法を試せます（広告視聴後に利用可能）：

1. カメラスキャン
2. PDF読み込み
3. URL入力（例：https://www.apple.com/legal/internet-services/itunes/jp/terms.html）
4. テキスト貼り付け

【APIキーについて】
OpenAI APIキーは、ビルド時に安全に注入されており、ソースコードには含まれていません。

【広告について】
無料プランでは、Google AdMobのリワード広告を使用しています。広告視聴後にAI解析が実行されます。広告が表示されない場合でも、デバッグ環境では解析を続行できるように実装しています。

【プライバシー】
• ユーザーデータは保存されません
• AI学習にも使用されません
• 解析後、データは速やかに削除されます
• OpenAI APIのデータ使用ポリシーに準拠しています

【サブスクリプション】
• スタンダード: ¥280/月（standard_monthly）
• プロ: ¥580/月（pro_monthly）

StoreKit Configuration Fileを使用してテストできます。

よろしくお願いいたします。
